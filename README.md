# Com-Pro_Game
# Introduction
กลุ่ม Life is game ของเราได้ทำการสร้างเกมส์ ซึ่งเป็นเกมส์แนว visual novel เป็นเกมส์แนว คล้ายๆกับการอ่านนิยาย ซึ่งเกมส์นี้พวกเราได้ทำการสร้างมาจากภาษา C ในการเขียน โดยมี library raylib เป็นตัวช่วยครับ
